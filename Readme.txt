How to install:
1. Move the whole 'RedWood' folder into your themes directory (.../themes).
2. Access the dashboard and click on 'Themes'.
3. Find the theme named 'RedWood' and select Apply.

-------------------------------------------------

This style works with the Categories2Menu plugin.

You can change your favicons and apple-touch-icons in the default.master.php in the views folder.
